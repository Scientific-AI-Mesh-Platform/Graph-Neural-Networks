Scientific Mesh Intelligence Core Data Structures. This repository is a part of the Scientific Mesh Intelligence open-source ecosystem.

## Overview
This package provides modular, high-performance, and scalable solutions for scientific mesh intelligence core data structures. Built for both researchers and industry practitioners, it emphasizes clean API design, extensive testing, and seamless integration with other SMI tools.

## Installation

You can install this package locally:
```bash
pip install -e .
```

## Developer Guide
We enforce strict typing, logging, and formatting.
- **Formatting**: `black` and `isort`
- **Typing**: `mypy`
- **Testing**: `pytest`

Run tests:
```bash
pytest tests/
```
